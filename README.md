# Diwali-Sales-Analysis-Using-Python
This project explores customer purchase behavior during Diwali season using Python. The goal is to understand which groups of customers contribute the most to sales and what factors influence spending patterns. The analysis uses Pandas, NumPy, Matplotlib and Seaborn for cleaning, summarizing and visualizing the data.
